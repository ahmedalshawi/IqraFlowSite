Quick answers

My audio download failed
Check you're on Wi-Fi and have storage free, then retry from Settings → Audio. Downloads resume where they stopped.
My streak reset unexpectedly
Streaks roll over at midnight local time. If a grace token was available it's used automatically - check Stats → Streak history, and email us if something still looks wrong.
Can I move my data to a new phone?
Yes - Settings → Backup creates an export file you can restore on any device. Shared khatms rejoin automatically with the same invite code.
How do I leave or delete a shared khatm?
Open the khatm → ⋯ menu → Leave. If you created it, you can delete it for everyone; members keep their personal reading history.
